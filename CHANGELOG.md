# Changelog

Todas as mudanças relevantes deste projeto serão documentadas neste arquivo.

O formato segue a ideia do [Keep a Changelog](https://keepachangelog.com/pt-BR/1.1.0/), e o versionamento deve seguir SemVer sempre que o projeto passar a publicar tags.

## [Não Lançado]

### Adicionado

- Projeto base em Laravel 13 com Jetstream e Livewire.
- Idioma `pt_BR` configurado.
- AdminLTE 4 integrado com assets separados dos assets padrão do Jetstream.
- Bootstrap, Font Awesome e OverlayScrollbars configurados via Vite.
- Tutoriais de criação do projeto e instalação do AdminLTE em `docs/`.
- README específico para uso como base clonável de novos projetos.
