# Roadmap

Este roadmap registra melhorias possíveis para o template Laravel AdminLTE.

## Próximos Passos

- Documentar fluxo recomendado para criar novos módulos administrativos.
- Criar exemplo de uma segunda página administrativa usando `layouts.adminlte`.
- Criar componentes Blade opcionais para cards, tabelas e botões do painel.
- Criar seeders de usuário administrador.

## Melhorias Futuras

- Adicionar suporte opcional a permissões e papéis de usuário.
- Incluir testes básicos para rotas principais.
- Adicionar workflow de CI no GitHub Actions.
- Criar release inicial com tag `v0.1.0`.

## Fora do Escopo Atual

- Regras de negócio específicas.
- Modelagem de banco de dados de domínio.
- Módulos administrativos prontos para um segmento específico.
- Integrações externas obrigatórias.
